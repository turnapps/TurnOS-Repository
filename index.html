TurnOS package repository

This is 

turnapps.github.io/TurnOS-Repository/
, A experimental Debian package repository for providing some TurnOS Packages.

Server

The server of this repository is hosted by GitHub Pages.

How to use

Add following text to sources.list, and run 

sudo apt update
:

  • deb [signed-by=/usr/share/keyrings/turnos-archive-keyring.gpg] https://turnapps.github.io/TurnOS-Repository/ ./

The GitHub repository is 

https://github.com/turnapps/TurnOS-Repository
.

NOTE: This is experimental repository, expect the bugs and report it to issues.
